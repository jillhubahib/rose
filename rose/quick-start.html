Quick Start

Install rose and rose-cli.

yarn add @rosepg/rose @rosepg/rose0cli --exact

Run the CLI to generate code from your existing database.

$(yarn bin)/rose --url postgresql://postgres:password@localhost:5432/yourdb --out generated/db

Build your queries using the DSL.

import { params, select } from "@rosepg/rose";
import { QStaff, StaffAllColumns, StaffRow } from "../generated/db/Staff";

interface Params {
	staffId: number;
}
const P = params<Params>();
const query = select(StaffAllColumns)
	.where(QStaff.staffId.eq(P.staffId))
	.finalise(P);

Create your node-pg client, and connect to the database.

import { Client } from "pg";
import { dummyClient } from "./fixtures";

const client = new Client({
	connectionString: "postgresql://postgres:password@localhost:5432/yourdb",
});
await client.connect();

Execute your query.

const rows = await query.execute(client, {
	staffId: 1,
});
if (rows.length > 0) {
	const row = rows[0];
	console.log(`Retrieved staff member ${row.firstName} ${row.lastName}`);
}

(Don’t forget to close the connection once you’re done!)
